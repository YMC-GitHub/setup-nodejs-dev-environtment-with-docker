# 如何使用此工具
此工具需要在docker环境已经安装，并且docker deamon已经运行下运行。


#### 管理篇
```sh
# 搭建
# 启动
./dev-start.sh
# 查看
./info.sh
# 执行任务
./dev-task.sh
# 停止
# 退出
# 删除
./dev=delete.sh
```


#### 配置篇

```sh
#查看
./print_conf.sh
#设置

```
